This is a test project to visualize COOL based information in ATLAS detector.

The project is based on JSF, and JBoss AS7.

The access to COOL system is done via a PL/SQL API which is stored in Oracle; all the queries need to know which COOL DB we want to access, and allow navigation into different schemas, nodes, tags.

Select on the COOL menu on the left the type of information that you want to retrieve.

The first item in the menu will allow to query global tags, using generic search string for schemas.

The second item allows to query a specific system information, together with some payload values.

The third item is conceived to search for nodes and tags over all schemas


Clicking on one the menu on the left will trigger a modification of the panel on the right.

In the query panel you can input parameters to define your search; there are several types of query inputs possible. For example when you search for global tags, you will have also a query sub-panel to search for differences among 2 global tags.

This project is a demo for COOL PL/SQL access, written by A.Formica
